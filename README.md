# 华容道

华容道游戏 cocos-creator 版本

## 目录结构

- assets
  - Scene
    - main.fire // 主场景
  - Script
    - Auto.ts // 自动行棋
    - Chess.ts // 棋子
    - Dialog.ts // 对白
    - HistoryPopup.ts // 历史进度弹层
    - Main.ts // 主场景
    - Map.ts // 地图
    - Popup.ts // 通用弹层（浅显特效）
